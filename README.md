# O que é Git | What is Git

🇧🇷  
O Git é um sistema de controle de versão que roda no computador.  
Ele salva o histórico do projeto e permite voltar para versões anteriores.

🇺🇸  
Git is a version control system that runs locally on your computer.  
It saves project history and allows you to go back to previous versions.

Exemplo:
Se você apagar algo sem querer, pode voltar.

---

# O que é GitHub | What is GitHub

🇧🇷  
GitHub é uma plataforma online onde guardamos os repositórios na nuvem.

🇺🇸  
GitHub is an online platform where repositories are stored in the cloud.

Permite:

- compartilhar código
- trabalhar em equipe
- contribuir em projetos

---

# Conceitos importantes | Important concepts

## Repositório | Repository

Pasta do projeto com histórico.

## Commit

Foto do projeto em um momento.

## Branch

Cópia da main para testar mudanças.

## Main

Linha principal do projeto.

## Merge

Juntar branch com main.

## Fork

Cópia de outro projeto.

## Pull Request

Pedido para juntar alterações.

---

# Comandos principais | Main commands

## git init

Inicia repositório
git init


## git add

Prepara arquivos


git add .


## git commit

Salva versão


git commit -m "primeiro commit"


## git branch

Define branch


git branch -M main


## git remote

Conecta com GitHub


git remote add origin LINK


## git push

Envia para GitHub


git push origin main


## git pull

Baixa do GitHub


git pull origin main


## git checkout

Trocar branch


git checkout main


Criar branch


git checkout -b nova-branch


## git merge

Juntar branch


git merge nova-branch

## 1 Criar pasta

Criar pasta:

README.md

## 5 Commit

git branch -M main


---

## 7 Criar repositório no GitHub

Criar no site.

Copiar link.

---

## 8 Conectar

Exemplo:

## 9 Enviar
